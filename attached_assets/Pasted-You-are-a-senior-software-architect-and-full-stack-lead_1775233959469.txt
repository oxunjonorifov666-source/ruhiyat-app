You are a senior software architect and full-stack lead engineer. Build a production-oriented monorepo project called “Ruhiyat” with clean architecture, scalable structure, strong security foundations, and clear separation of concerns.

PROJECT OVERVIEW:
“Ruhiyat” is a digital mental wellness ecosystem with 3 major products:
1) Superadmin web panel
2) Administrator web panel
3) Mobile app for end users (Android-first, Play Market ready, APK/AAB compatible)

IMPORTANT PRODUCT LOGIC:
- Superadmin panel is for managing the mobile ecosystem and overall platform
- Administrator panel is for education center owners / managers
- Mobile app is for users
- Web panels run in browser
- Mobile app must be built as a real React Native mobile app, not a fake mobile-responsive web page
- Mobile app architecture must be ready for Android APK/AAB release and future Play Market publishing

USE THIS TECH STACK:
- Monorepo
- Frontend web: Next.js + TypeScript + Tailwind CSS + shadcn/ui
- Mobile: React Native + Expo + TypeScript
- Backend API: NestJS + TypeScript
- Database: PostgreSQL
- ORM: Prisma
- Authentication: JWT access token + refresh token
- Password hashing: bcrypt
- Validation: class-validator / zod where appropriate
- API style: REST
- State management: use clean maintainable approach
- File storage abstraction ready
- Environment variables for all secrets
- Strict TypeScript everywhere

ARCHITECTURE REQUIREMENTS:

- Make the project modular, scalable, and future-proof
- Avoid hardcoding secrets
